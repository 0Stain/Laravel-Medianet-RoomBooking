## Installation
* Clone the repositore using **git clone https://github.com/0Stain/Laravel-Medianet-RoomBooking.git**
* Copy **.env.example**, rename it **.env** and change to your variables
* Run these commands
``` bash
composer update
```
``` bash
php artisan key:generate
```
``` bash
php artisan migrate
```
``` bash
php artisan db:seed
```
